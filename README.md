## ğŸš€ Getting Started

### Prerequisites

```bash
node >= 16.x
npm >= 8.x
expo-cli
```

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/yourusername/vula-moto.git
cd vula-moto
```

2. **Install dependencies**
```bash
npm install
```

3. **Configure Firebase**
```bash
# Add your Firebase configuration to firebase.config.js
cp firebase.config.example.js firebase.config.js
```

4. **Start the development server**
```bash
npm start
# or
expo start
```

5. **Run on device**
```bash
# iOS
npm run ios

# Android
npm run android
```
